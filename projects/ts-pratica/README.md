# Prática TypeScript (tipagem e OOP)

Arquivos:
- `index.ts`: código-fonte em TypeScript com as duas práticas (funções/objetos tipados e classes com herança/encapsulamento).
- `index.js`: versão compilada manualmente para facilitar execução sem build.

Como executar:
- Em TypeScript: `npm install` e `npm run build:ts` (saída em `dist/`, se preferir).
- Ou simplesmente: `node index.js` nesta pasta `projects/ts-pratica/` para ver a saída no terminal.

O código imprime os resultados das funções (soma, multiplicar, criarPessoa, descreverCarro) e das classes Funcionario/Gerente/Diretor usando o nome "Kauai Rocha".
